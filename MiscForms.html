חומר עזר לתרגול

 

טפסים

   עד כה עסקנו בכתיבת דפים שבאמצעותם יכול היה המשתמש לדפדף בין דפי האתר ולהגיע דרכם לאתרים אחרים או לכתובות דואר. המשתמש לא שינה דבר באתר ולא העביר דבר. עכשיו אנו מגיעים לקטע האינטראקטיבי - האפשרות להעביר נתונים ממחשב הלקוח אל מחשב השרת. דבר זה נעשה באמצעות טפסים והתגית הבסיסית היא <FORM>.

   כדי שהתוכן שמכניס המשתמש באתר אכן יגיע אל השרת נחוץ דבר נוסף: פרוטוקול תקשורת להעברת נתונים מן הלקוח לשרת לצורך עיבודו. פרוטוקול זה קרוי COMMON GATEWAY INTERFACE או בקיצור CGI.

   לטופס יש ארבעה תגים בסיסיים: FORM, TEXTAREA, SELECT, INPUT.

<FORM> מגדיר את תחום הטופס. יש לו שלוש תת תגיות:

ACTION  שאומרת לאן ישלח הטופס (URL=ACTION).     

ENCTYPE שמאפיין את המדיה לקידוד המידע בטופס

תחביר:             <URLECNCODED-FORM-WWW-X/APPLICATION=FORM ENCTYPE>

METHOD מגדיר כיצד ישלח המידע לכתובת שצוינה ב-ACTION.

תחביר:        VALUE=METHOD כאשר הערך הוא או GET - שלוף את המידע מהמסמך הנוכחי

או POST - הדף הנוכחי ישלח את המידע אל תוכנת עיבוד (תסריט CGI כלשהו).

תחת <FORM> יבוא התג <TEXTAREA> המקצה שטח כתיבה להכנסת הנתונים. תת התגיות החשובות שלו הן:

VALUE=NAME הקצאת שם לטקסט המוכנס.

N=ROWS הקצאת מספר השורות המיועדות לקליטת הטקסט (N מייצג את מספר השורות)

N=COLS הקצאת רוחב האזור לקליטת הטקסט (N מייצג מספר תוים).

VALUE=WRAP טיפול בגלישת מלים (האפשרויות הן OFF, VIRTUAL ו-PHYSICAL).

 

תגית <SELECT>

תגית SELECT מאפשרת למשתמש לבחור אפשרות אחת מתוך רשימת אפשרויות שכותב האתר מציג בפניו במקום להכניס טקסט משלו. לרשימת SELECT ניתן לתת שם באמצעות תת תגית NAME. ניתן לפתוח את הרשימה עם הצגתה באמצעות תת תגית SIZE  הקובעת כמה אפשרויות יוצגו מכלל האפשרויות. תת תגית MULTIPLE מאפשרת בחירה של כמה אפשרויות מן הרשימה (רשימה פרושה) את האפשרויות עצמן מציגים עם תגית <OPTION> המקוננת בתוך תגית <SELECT>. תת תגית SELECTED  קובעת איזו אפשרות (או אפשרויות במקרה של MULTIPLE) תסומן מראש ברשימה. תת תגית VALUE מאפשרת מתן ערך מספרי לאופציה שישלח למקבל הטופס אם המשתמש יבחר באופציה.

 

תגית <INPUT>

בניגוד ל <TEXTAREA> הקולט מלל חופשי. בתגית INPUT יתן להגדיר במדויק איזה סוג מידע מתבקש המשתמש להכניס. תת תגית TYPE מגדירה את סוג השדה (TEXT, PASSWORD, CHECKBOX, RADIO, BUTTON וכו')
